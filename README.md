#  CodSoft Java Internship Projects

This repository contains all the projects completed as part of the **CodSoft Internship (Java Development)**.

---

##  Projects Included

###  1. Number Guessing Game

An interactive game where the user guesses a randomly generated number based on selected difficulty levels.

**Features:**

* Multiple difficulty levels (Easy, Medium, Hard)
* Limited attempts system
* Real-time feedback (Too High / Too Low)
* Restart and Give Up options

**Tech Stack:**

* HTML
* CSS
* JavaScript

---

###  2. Student Grade Calculator

A web-based application that calculates total marks, average percentage, and grade.

**Features:**

* Add multiple subjects dynamically
* Input marks for each subject
* Calculates total, average, and grade
* Color-coded grade display

**Tech Stack:**

* HTML
* CSS
* JavaScript

---

###  3. Currency Converter

A real-time currency converter using exchange rate API.

**Features:**

* Convert between multiple currencies (USD, INR, EUR, GBP)
* Fetches real-time exchange rates
* Simple and user-friendly UI

**Tech Stack:**

* HTML
* CSS
* JavaScript
* API (Exchange Rate API)
